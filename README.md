### Instalação

```bash
$ yarn
```

### Executando

```bash
# Iniciando aplicação na porta 3000
$ yarn dev
```

### Testando

```bash
# Execuntando todos testes
$ yarn test

# Executando conjunto de testes
$ yarn test src/__test__/{folder}/{fileName}.spec.tsx

# Executando teste específico
$ yarn -t "testName"

```
